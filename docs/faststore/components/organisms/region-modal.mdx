It's a [Modal](https://developers.vtex.com/docs/guides/faststore/molecules-modal) with some region configurations. The component is part of [Regionalization](https://developers.vtex.com/docs/guides/faststore/feature-regionalization) feature.

## Import

Import the component from [@faststore/ui](https://developers.vtex.com/docs/guides/faststore/components-index)

```tsx
import { RegionModal } from '@faststore/ui'
```

### Import Styles into your FastStore project

To apply the styles of this component in your FastStore project, import the following into your stylesheet:

```scss
@import '@faststore/ui/src/components/organisms/RegionModal/styles.scss';
```

Follow the instructions in the [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles) tutorial.

## Usage

<Tabs items={['Example', 'Code']} defaultIndex="0">
  <Tab>
    <OverviewSection>
      <UIProvider>
        <RegionModalUsage />
      </UIProvider>
    </OverviewSection>

  </Tab>

    ```tsx
    const { modal, openModal } = useUI()
    const [input, setInput] = useState('')
    return (
      <>
        <Button variant="primary" onClick={() => openModal()}>
          Open Region Modal
        </Button>
        {modal && (
          <RegionModal
            fadeOutOnSubmit={true}
            inputValue={input}
            onInput={(e) => setInput(e.currentTarget.value)}
          />
        )}
      </>
    )
    ```

  </Tab>
