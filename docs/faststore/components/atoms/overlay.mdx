Overlays provide contextual information on top of the current layer of the interface. They can be interruptive like modals or augmenting like popovers and tooltips.

## Overview

```tsx
function Component() {
  const styles = {
    container: {
      position: 'relative',
      height: '8rem',
    },
    overlay: {
      position: 'absolute',
    },
  }
  return (
    <div style={styles.container}>
      A content outside me
      <Overlay style={styles.overlay} />
    </div>
  )
}
```

## Import

Import the component from [@faststore/ui](https://developers.vtex.com/docs/guides/faststore/components-index)

```tsx
import { Overlay } from '@faststore/ui'
```

### Import Styles into your FastStore project

To apply the styles of this component in your FastStore project, import the following into your stylesheet:

```scss
@import '@faststore/ui/src/components/atoms/Overlay/styles.scss';
```

Follow the instructions in the [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles) tutorial.
